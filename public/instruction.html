Let's Play Connect4!

Go to this link to access the game. This page is the backend server.
the game rule is, well... just like Connect4!
Your task is to create a bot or Artificial Intelligence that can play Connect4 automatically. Your bot will play against a bot that you select. The stronger your opponent, the higher the score that you will get!
Here are pre-defined methods with default code for both Python and JavaScript that you have to complete:

Python Code:

def move(board):
    # start your code here
    # must return integer which is the column index
    return randint(0,6);

JavaScript Code:


function  move(board){
    # start your code here
    # must return integer which is the column index
    return Math.floor(Math.random()*6)
}

Here is the Data Structure of the Board to help you write your bot, it is a 2D array - 6 columns and 7 rows:


[ ["_", "_" , "_" , "_" , "_" , "_" , "_" ],

Here are some rules that you should follow:

  • in every step, the bot should return an integer, which is the column index
  • you must login using facebook first in order to submit code

Last but not least, Happy Playing!!!

POST /submit_bot

Request:

type: application/json

values:

{
    "bot": "actual code written by the user",
    "lang": "python or js",
    "userid": "user id or facebook id"
}

Response:

Syntax Error (JSON):

{"success": false, "error": "invalid syntax"}

Bad Move (JSON):

{"success": false, "error": "bot cause invalid move"}

Success (JSON):

{
    "success": true,
    "error": null,
    "bot": {
        'code': code,
        'id': 0,
        'lang': lang,
        'name': "Bot0",
        'score': 800,
        'win': 0,
        'lose': 0
    }
}
